# check-socketio

## Default

- BE Port: `3005`
- FE Port: `5005`
- BE URL: `http://localhost:3005/`
- FE URL: `http://localhost:5005/`

## Setup BE

### Install

```bash
$ cd be
$ npm install
$ cp .env.sample .env
$ npm start
```

## Setup FE

### Install

```bash
$ cd fe
$ npm install
$ npm start
```

## Execution
- Open 2 browser windows
- Open `http://localhost:5005/chat` on both browsers
- Happy chatting
